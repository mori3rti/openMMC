Writes a number of bytes, at most {\itshape size}, from the memory area pointed to by {\itshape buffer}. 

If {\itshape buffer} is zero then \hyperlink{write_8c_ad0145c89238a86c1b59c1d4a62bf29d8}{\-\_\-\-\_\-write} performs flushing of internal buffers, if any. In this case, {\itshape handle} can be {\ttfamily -\/1} to indicate that all handles should be flushed.

\end{DoxyParams}
